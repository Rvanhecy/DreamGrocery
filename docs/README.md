# 🏪 梦境杂货铺 (Dream Grocery)

本项目是一款基于 Gemini 2.5 Flash 引擎开发的治愈系 AI 交互小游戏。

### 📦 如何运行
1. **解压**本压缩包到桌面。
2. **双击**根目录下的 `index.html`。
3. **环境说明**：请确保电脑处于联网状态。项目已内置加密的演示 API Key，无需任何后端配置。

### 🌟 核心特色
- **Vibe Coding 实践**：利用自然语言调教 AI 情绪，打造有温度的交互体验。
- **多模态集成**：实时生成像素艺术图像、感性文字叙事及 TTS 情感配音。
- **离线保底**：内置 Static Failback 机制，确保在网络波动时依然可玩。
