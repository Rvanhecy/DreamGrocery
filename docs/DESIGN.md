# 🎨 设计逻辑与技术架构

### 1. Vibe Coding 核心理念
本项目采用 **Vibe Coding (感性驱动开发)** 范式。核心逻辑不再是死板的代码分支，而是通过对 Gemini 的 `System Instruction` 进行深度语境约束：
- 将 AI 设定为“温暖而略带忧郁的杂货铺主理人”。
- 实现了根据不同游戏模式自动切换“人称视角”的动态叙事逻辑。

### 2. 技术优势
- **多模态闭环**：单次请求同步完成语义分析、图像生成指令和音频合成。
- **工程安全性**：采用立即执行函数 (IIFE) 封装敏感凭证，并进行 Base64 混淆处理。
- **本地化存储**：利用浏览器 `localStorage` 实现成就与梦境册的本地持久化。

### 3. UI/UX 风格
采用了 **Dreamcore (梦幻核)** 与像素风结合的视觉基调，旨在通过复古的低保真感抵消 AI 的“塑料味”。
